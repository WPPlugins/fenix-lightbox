== Description ==

Fenix Lightbox make shortcodes to use in posts or pages, in the place you need. Use it to open images or pop ups using the lightbox effect "facebox", this script is development by famspam.com/facebox

== Installation ==

1. Upload the folder <code>fenix-lightbox</code> to the <code>/wp-content/plugins/</code> directory
2. Activate the plugin in the Plugins menu in your WordPress.
3. Use <code>[lbfenix img="file.jpg"]link name[/lbfenix]</code> for open a image widht the Lightbox efect or use <code>[lbfenix id="123" title="name of link"] Text in lightbox [/lbfenix]</code> for open a text widht the Lightbox efect.
